# Поддерживаемые форматы JSON

Система автоматически определяет тип JSON файла и конвертирует его в текст для обучения.

---

## 1️⃣ Формат "tasks" (Задачи из OCR)

**Используется для:** Задачи по химии/физике/математике из учебников

**Структура:**
```json
{
  "dataset_name": "chemistry_tasks",
  "created_at": "2026-05-31T13:59:22",
  "total_tasks": 7,
  "tasks": [
    {
      "question": "Какова молярная масса H₂O?",
      "solution": "M(H₂O) = 2·1 + 16 = 18 г/моль",
      "answer": "18 г/моль",
      "category": "Задача",
      "difficulty": "medium",
      "keywords": ["молярная масса", "вода"]
    }
  ]
}
```

**Ключевое поле:** `"tasks"` (массив задач)

**Конвертируется в:**
```
### Задача 1 (Задача)

Вопрос: Какова молярная масса H₂O?

Решение:
M(H₂O) = 2·1 + 16 = 18 г/моль

Ответ: 18 г/моль
```

**Источник:** OCR приложение (`ocr_app.py`)

---

## 2️⃣ Формат "interview" (Вопросы для интервью)

**Используется для:** Вопросы для технических интервью (Python, JavaScript и др.)

**Структура:**
```json
{
  "items": [
    {
      "topic": "последовательности",
      "source_type": "import_api",
      "content": "Вопрос: Что такое последовательность\n\nЭталонный ответ:\nПоследовательностью в Python называется..."
    }
  ]
}
```

**Ключевое поле:** `"items"` (массив вопросов)

**Конвертируется в:**
```
## Тема: последовательности

Вопрос: Что такое последовательность

Эталонный ответ:
Последовательностью в Python называется...

---
```

**Источник:** Репозитории с вопросами для интервью (например, `yakimka/python-interview-questions`)

**Пример файла:** `data/python_interview_questions.json` (148 вопросов, 191,575 символов)

---

## 3️⃣ Формат "forum" (Форумные сообщения)

**Используется для:** Диалоги с форумов, чатов

**Структура:**
```json
{
  "User": "JohnDoe",
  "messages": {
    "Тема 1": [
      {"author": "User1", "text": "Вопрос..."},
      {"author": "User2", "text": "Ответ..."}
    ]
  }
}
```

**Ключевое поле:** `"messages"` или `"User"`

---

## 🚀 Использование

### В Streamlit UI:

1. Скопируйте JSON файл в `pytorch_llm/data/`
2. Откройте вкладку **🎓 Обучение**
3. Выберите файл из выпадающего списка
4. Система автоматически определит формат и конвертирует

### В CLI:

```bash
poetry run python train.py \
  --data data/python_interview_questions.json \
  --model-name python_qa_model \
  --epochs 10 \
  --batch-size 32 \
  --context-len 128
```

## 📊 Статистика форматов

| Формат | Avg размер | Типичное кол-во | Use case |
|--------|-----------|-----------------|----------|
| tasks | 2-10 KB | 5-20 задач | Репетитор по предмету |
| interview | 50-200 KB | 50-200 вопросов | Подготовка к интервью |
| forum | 10-100 KB | 20-100 сообщений | Диалоговый бот |

---

## 💡 Рекомендации

**Для маленьких датасетов (<10 KB):**
- Context Length: 64-128
- Batch Size: 8-16
- Epochs: 10-20

**Для средних датасетов (10-100 KB):**
- Context Length: 128-256
- Batch Size: 16-32
- Epochs: 5-15

**Для больших датасетов (>100 KB):**
- Context Length: 256-512
- Batch Size: 32-64
- Epochs: 3-10

   ```python

       pass
   ```

2. Обновить `detect_json_type()`:
   ```python
   if 'your_key' in data:
       return 'your_format'
   ```

3. Добавить в `load_data()`:
   ```python
   elif json_type == 'your_format':
       text = convert_YOUR_FORMAT_json_to_text(data_path)
   ```
