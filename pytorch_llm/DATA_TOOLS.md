# 🛠️ Инструменты работы с данными

## Новые возможности в UI (раздел "📁 Данные")

### Пункт 5: Конвертер PDF → TXT

**Что делает:**
- Загружает PDF файл через браузер
- Сохраняет в `data/input_pdf/`
- Конвертирует в чистый текст
- Сохраняет результат в `data/`

**Как использовать:**
1. Нажмите "Browse files" и выберите PDF книгу
2. Включите/выключите "Очистить текст" (рекомендуется ✅)
3. Укажите имя выходного файла (по умолчанию: `book_converted.txt`)
4. Нажмите "🔄 Конвертировать PDF"
5. Получите статистику: страницы, символы, слова, размер
6. Проверьте превью первых 500 символов

**Зачем очистка текста:**
- ❌ Без очистки: разрывы строк внутри слов, лишние пробелы
- ✅ С очисткой: правильные параграфы, убраны артефакты PDF

---

### Пункт 6: Объединитель датасетов

**Что делает:**
- Объединяет несколько `.txt` файлов в один
- Добавляет заголовки с именами файлов
- Разделяет датасеты специальным разделителем
- Анализирует итоговый датасет (размер, язык, статистика)

**Как использовать:**
1. Выберите 2+ файла из выпадающего списка
2. Включите "Добавить заголовки" (рекомендуется)
3. Настройте разделитель (по умолчанию: `\n\n---\n\n`)
4. Укажите имя выходного файла (по умолчанию: `combined_dataset.txt`)
5. Проверьте превью выбранных файлов
6. Нажмите "🔗 Объединить датасеты"
7. Получите анализ:
   - Размер (MB), символы, слова, строки
   - Процент кириллицы/латиницы
   - Рекомендации по языку

**Индикаторы качества:**
- ✅ **Кириллица >80%** — хороший русский датасет
- ℹ️ **Латиница >80%** — английский датасет
- ⚠️ **Смешанный** — модель может путаться

## Типичный workflow для химии

### Шаг 1: Собрать PDF книги
Скачайте легальные PDF учебники по химии (5-10 штук)

### Шаг 2: Конвертировать каждую книгу
1. Откройте "Пункт 5: Конвертер PDF → TXT"
2. Загрузите `chemistry_book1.pdf`
3. Конвертируйте → получаете `chemistry_book1_converted.txt`
4. Повторите для всех книг

### Шаг 3: Объединить в один датасет
1. Откройте "Пункт 6: Объединить датасеты"
2. Выберите все конвертированные `.txt` файлы:
   - `chemistry_school_belarus.txt` (44 KB)
   - `chemistry_book1_converted.txt`
   - `chemistry_book2_converted.txt`
   - ...
3. Объедините → получаете `chemistry_combined.txt` (5-10 MB)
4. Проверьте анализ: должно быть >90% кириллицы

### Шаг 4: Обучить модель
1. Перейдите в "🎓 Обучение"
2. Dataset: выберите `chemistry_combined.txt`
3. Tokenizer: `tiktoken (cl100k_base)`
4. Конфигурация: `small` (для 5-10 MB)
5. Epochs: 10-15
6. Начать обучение!

## CLI альтернативы (если нужно)

### PDF → текст (командная строка)
```bash
cd pytorch_llm/utils

# Один файл
python pdf_to_text.py book.pdf output.txt

# Вся директория
python pdf_to_text.py --dir ~/pdfs/ --output-dir ../data/texts/
```

### Объединение (командная строка)
```bash
cd pytorch_llm/utils

# Анализ одного файла
python merge_datasets.py --analyze ../data/chemistry_school_belarus.txt

# Объединение нескольких файлов
python merge_datasets.py file1.txt file2.txt file3.txt --output combined.txt

# Объединение по паттерну
python merge_datasets.py --dir ../data/ --pattern "chemistry_*.txt" --output chemistry_all.txt
```

## Технические детали

### convert_pdf_to_text()
- **Библиотека:** pypdf 6.12.2
- **Очистка:** удаляет множественные пробелы, восстанавливает параграфы, убирает пустые строки
- **Возврат:** dict с `pages`, `chars`, `words`, `size_kb`

### merge_text_files()
- **Разделитель:** настраиваемый (по умолчанию `\n\n---\n\n`)
- **Заголовки:** добавляет `=== [filename] ===` перед каждым датасетом
- **Анализ:** подсчитывает символы, слова, строки, процент кириллицы/латиницы
- **Возврат:** dict с `output_path` и `analysis`

### analyze_dataset()
- **Метрики:** размер (bytes/KB/MB), символы, слова, строки
- **Язык:** подсчитывает кириллицу и латиницу отдельно
- **Использование:** автоматически вызывается при объединении

## FAQ

**Q: Какой размер PDF поддерживается?**  
A: Ограничений нет, но для больших файлов (>100 MB) конвертация займет больше времени.

**Q: PDF с картинками/формулами конвертируется?**  
A: Извлекается только текст. Картинки и сложные формулы игнорируются.

**Q: Какой минимальный размер датасета для обучения?**  
A: Минимум 1 MB, рекомендуется 5-10 MB для small модели.

**Q: Можно ли объединять файлы на разных языках?**  
A: Да, но модель может путаться. Лучше держать один язык >80%.

**Q: Нужно ли вручную чистить текст после конвертации?**  
A: Обычно нет, автоочистка справляется. Но можно открыть файл и проверить.
