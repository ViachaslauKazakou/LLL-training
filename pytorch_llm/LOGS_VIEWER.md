# 📊 Вкладка "Логи" — документация

## Обзор

Новая вкладка **"📊 Логи"** позволяет просматривать логи в реальном времени прямо в браузере, без необходимости открывать терминал.

**Путь к логам:** `/Users/Viachaslau_Kazakou/Work/LLM-learn/logs/`

---

## Функции

### 📄 Выбор файла лога

Dropdown с доступными файлами:
- `app.log` — логи Streamlit приложения
- `data.log` — логи загрузки и обработки данных
- `inference.log` — логи генерации текста
- `training.log` — логи обучения модели ⭐

---

### 🔍 Фильтр по уровню

Фильтрация логов по важности:
- **ALL** — показать все логи (по умолчанию)
- **INFO** — только информационные сообщения
- **WARNING** — только предупреждения
- **ERROR** — только ошибки
- **DEBUG** — только отладочные сообщения

---

### ▶️ Автообновление

**Кнопка "▶️ Старт" / "⏸️ Стоп":**
- Включает/выключает автообновление логов
- Обновление происходит **каждые 5 секунд**
- Показывает время последнего обновления (X сек. назад)
- Использует `st.session_state` для сохранения состояния

**Как это работает:**
1. Нажмите "▶️ Старт"
2. Вкладка будет автоматически перезагружаться каждые 5 секунд
3. Новые строки лога появятся в интерфейсе
4. Нажмите "⏸️ Стоп" чтобы остановить

---

## 🔧 Технические детали

### Автообновление

Использует `st.session_state` для хранения состояния:
```python
st.session_state.logs_auto_refresh = True/False
st.session_state.logs_last_refresh = timestamp
```

Проверка времени:
```python
if current_time - st.session_state.logs_last_refresh >= 5:
    st.session_state.logs_last_refresh = current_time
    st.rerun()  # Перезагрузка вкладки
```

### Чтение файлов

- Читает весь файл: `readlines()`
- Применяет фильтр по уровню
- Берет последние N строк: `lines[-num_lines:]`
- Отображает в `st.text_area()` для возможности копирования

### Цветная подсветка

Использует:
- Markdown жирный текст: `**text**`
- Emoji префиксы: 🔄 ✅ ❌ ⚠️ ℹ️
- Regex для поиска паттернов: `re.sub(r'(loss=[\d\.]+)', r'**\1**', line)`

## ⚠️ Важные замечания

### Производительность
- Файлы >10 MB могут загружаться медленно
- Рекомендуется ограничить количество строк до 500
- Автообновление каждые 5 секунд может нагружать систему

### Ограничения
- Логи читаются целиком, не инкрементально
- Для очень больших файлов (>100 MB) используйте CLI: `tail -f logs/training.log`
- Автообновление работает только пока вкладка активна

### Рекомендации
- Для мониторинга обучения: используйте автообновление
- Для анализа ошибок: отключите автообновление, используйте фильтр ERROR
- Для копирования логов: выделите текст в text_area и Cmd+C
