# Как обучить Mini LLM на стиль конкретного человека

## Требования к корпусу

Для минимально осмысленного результата нужно:

1. **Объём**: 100–500 предложений (чем больше, тем лучше)
2. **Разнообразие**: разные темы, но узнаваемый стиль
3. **Характерные особенности**:
   - Любимые слова и выражения
   - Эмодзи / знаки препинания
   - Структура предложений
   - Сленг / мемы

## Ограничения Mini LLM

| Параметр | Mini LLM | Реальный GPT |
|----------|----------|--------------|
| Словарь | 50–500 слов | 50K+ токенов |
| Контекст | 4–8 слов | 2K–8K токенов |
| Параметры | ~6K | 117M–175B |

**Что модель НЕ сможет:**
- Понимать смысл и отвечать по теме
- Сохранять логику диалога
- Генерировать длинные связные тексты

**Что модель МОЖЕТ:**
- Копировать характерные n-граммы (биграммы, триграммы)
- Повторять частые слова и обороты
- Имитировать "feel" стиля (эмодзи, междометия, знаки)

## Пример структуры corpus.json

```json
{
  "sentences": [
    "лол это прям топ 😂",
    "да ладно серьёзно?",
    "ну типа я хз честно",
    "бро это имба",
    ...
  ]
}
```

## Процесс

1. Собери 100+ фраз/реплик человека
2. Раздели на короткие предложения (4–10 слов)
3. Сохрани в `corpus_style.json`
4. Обучи модель:
   - `d_model=32`
   - `context_len=6`
   - `n_epochs=500–1000`
   - `lr=0.005–0.01`
5. Генерируй с `repetition_penalty=1.2–1.5`

## Ожидания

- ✅ Узнаваемые фразы-шаблоны
- ✅ Характерная лексика
- ✅ "Атмосфера" стиля
- ❌ Смысловые ответы на вопросы
- ❌ Логика и контекст диалога
- ❌ Длинные связные тексты

Для **реального** чат-бота в стиле человека нужна fine-tuned GPT-2/GPT-3 на ~10K+ реплик.
