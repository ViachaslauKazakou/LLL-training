# Быстрый старт: обучение на своих данных

## Шаг 1: Подготовьте текстовый файл

Создайте `.txt` файл с вашими фразами. Формат:
- Одна фраза на строку
- Или предложения через точку/перенос строки
- Минимум 50-100 фраз для хорошего результата

**Пример** (`my_style.txt`):
```
привет как дела
отлично спасибо
что нового у тебя
ничего особенного работаю
```

## Шаг 2: Создайте корпус

### Вариант A: CLI
```bash
cd mini_llm/v3
poetry run python cli.py
# → [P] Подготовить корпус из текста
# Путь к файлу: my_style.txt
# Сохранить как: corpus_my_style.json
```

### Вариант B: Скрипт
```bash
cd mini_llm/v3
poetry run python prepare_corpus.py my_style.txt corpus_my_style.json \
  --min-words 2 --max-words 12
```

## Шаг 3: Обучите модель

### CLI
```bash
poetry run python cli.py
# → [1] Загрузить корпус (corpus_my_style.json)
# → [2] Обучить (300-800 эпох, lr=0.01)
# → [6] Сохранить модель
```

### Программно
```python
from model import MiniLLM

llm = MiniLLM(d_model=32, context_len=6)
llm.load_corpus('corpus_my_style.json')
llm.train(n_epochs=500, lr=0.01)
llm.save('saved/my_style.npz')
```

## Шаг 4: Генерируйте текст

```python
llm.generate(
    'привет как', 
    n_new=6, 
    temperature=0.6,
    repetition_penalty=1.3
)
# → 'привет как дела отлично спасибо тебе большое'
```

## Рекомендации по параметрам

| Размер корпуса | d_model | context_len | epochs | lr |
|-----------------|---------|-------------|--------|----|
| 50-100 фраз | 24 | 5 | 300 | 0.01 |
| 100-300 фраз | 32 | 6 | 500 | 0.008 |
| 300+ фраз | 40-48 | 6-8 | 800 | 0.008 |

**Temperature**: `0.5-0.7` для более последовательного текста, `0.8-1.0` для разнообразия

**Repetition penalty**: `1.2-1.5` чтобы избежать зацикливания

## Частые проблемы

**Зацикливание** → увеличьте `repetition_penalty` до 1.4-1.5

**Случайный текст** → уменьшите `temperature` до 0.5-0.6

**Модель предсказывает одно и то же** → слишком высокий `lr` или переобучение:
- Уменьшите `lr` до 0.005-0.008
- Увеличьте размер корпуса
- Увеличьте `d_model` и `context_len`
