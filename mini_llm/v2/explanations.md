Это mini_llm.py версия 2 — та же модель, но теперь с обучением. Главное отличие от первого файла — добавлен полный цикл backward pass и SGD.

Что нового по сравнению с v1
1. Больше данных и эмбеддинг D=16
Корпус расширен (повторяется несколько раз), размер эмбеддинга удвоен с 8 → 16 для лучшей ёмкости модели.

2. Backward pass (ручное дифференцирование)
Функция loss_and_grads() реализует обратное распространение ошибки вручную — без autograd:
Loss (cross-entropy)
  → dlogits = probs - one_hot(target)
  → dWo, dlast
  → dctx
  → dA (веса attention), dVm
  → dWv, dWq, dWk
  → dX (градиент по входам)
  → dE (градиент по эмбеддингам)

Это показывает, как именно градиент течёт через каждый слой.

3. Обучение — SGD + gradient clipping
Gradient clipping предотвращает "взрывной" рост градиентов — типичная проблема в трансформерах.

```
norm = np.linalg.norm(g)
if norm > 1.0:
    g = g / norm   # обрезаем слишком большие градиенты
param -= lr * g    # обновляем параметры
```

4. Проверка результатов
После обучения выводятся top-3 предсказания с вероятностями для тестовых контекстов — можно увидеть, что модель реально выучила паттерны из корпуса.
